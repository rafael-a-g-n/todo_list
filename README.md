# My Todo List

A simple and stylish to-do list application built with React. This project allows users to create and manage their tasks with a clean and intuitive interface.

## Features

- **Add and Delete Todo Lists:** Easily create new to-do lists with custom headings and remove them when they are no longer needed.
- **Add and Delete List Items:** Add individual items to each to-do list and remove them once they are completed.
- **Responsive Design:** The application is fully responsive and works seamlessly on both desktop and mobile devices.
- **Enter to Submit:** Add new headings and list items by pressing the Enter key for a faster workflow.

## Technologies Used

- **React:** A JavaScript library for building user interfaces.
- **Vite:** A fast and lightweight build tool for modern web development.
- **ESLint:** A pluggable and configurable linter tool for identifying and reporting on patterns in JavaScript.

## Setup and Usage

To get started with this project, follow these steps:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/rafael-a-g-n/todo_list.git
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Run the development server:**

   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
5. **Preview the production build:**
   ```bash
   npm run preview
   ```
